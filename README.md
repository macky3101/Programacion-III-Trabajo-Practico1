# Tragos y Cócteles

Este sitio web presenta información sobre tragos, incluyendo recetas y horarios de servicio.

## Páginas

- **index.html**: Página principal con introducción a los tragos.
- **receta.html**: Receta de Mojito con ingredientes y pasos.
- **horario.html**: Horarios de servicio de lunes a domingo.

## Instalación

1. Clona el repositorio.
2. Abre index.html en un navegador.

## GitHub Pages

Para publicar en GitHub Pages:

1. Crea un repositorio en GitHub.
2. Sube este código al repositorio.
3. Ve a Settings > Pages, selecciona la rama main y la carpeta raíz.
4. El sitio estará disponible en https://tuusuario.github.io/nombre-repo/

## Tecnologías
